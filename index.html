Sobre Mi

Programador Web en linea de carrera con búsqueda constante de la mejora continua

Soy un hombre de 41 años, con una familia hermosa, gusto por la lectura y series de espionaje. Bachiller de la Universidad Nacional Mayor de San MArcos y con estudios de Post Grado en Psicología Organizacional.

Inicié en la programación por curiosidad y necesidad. Al inicio, tenía miedo de no poder entender nada, pues soy de una profesión de letras, sin embargo, pude romper el prejuicio que me limitaba y ahora no paro de aprender y buscar siempre nuevas formas de mejorar el código, nuevas formas de solucionar problemas y mejores animaciones.

Actualmente mi experiencia desarrollando código es en FrontEnd e iniciando BackEnd, así como algunas librerías. Una de las librerías que estoy aprendiendo por mi cuenta es React Three Fiber, para efectos 3D espectaculares.

Proyectos

Estos diferentes proyectos son parte de mi aprendizaje, que van desde la maquetación y diseño responsive, el uso de APIs, manejos de CRUD y algunas librerías.

Diseño Web

Este es el primer proyecto de maquetación y diseño web. Uso de Html y CSS, aún sin librerías, pero iniciando en el responsive.

Proyecto

Frases Random

Aplicación web desarrollada en React y Vite. El objetivo es que con cada Click en un Botón, se muestre de manera random una frase célebre, con su autor y de igual manera cambie el fondo. Para este caso he usado la librería TsParticles y react-tsparticles.

Proyecto

App de Clima

Proyecto de React que muestra el clima en base a la geolocalización del usuario, en cualquier país del mundo. Adicionalmente puede buscar otra ciudad y hacer la comparación. Se usa la Api Open Weather Map y una petición GET.

Proyecto

Rick And Morty

Proyecto realizado en React y Vite, usando la Api Rick And Morty. Tiene un botón que trae de manera aleatoria personajes por categoría. También se puede ingresar el orden numérico de los personajes. Adicionalmente tiene paginación para aquellas categorías de gran cantidad de personajes.

Proyecto

CRUD de Usuarios

Proyecto de Reacto y Vite con diseño responsive. Usa una Api propia de Academlo. Se desarrolla un CRUD de usuarios. Se empleó la librería de axios, Sweet Alert2, React Bootstrap y React Hook Form. PErmite poder crear, editar y eliminar usuarios con su respectivo alert de diseño.
